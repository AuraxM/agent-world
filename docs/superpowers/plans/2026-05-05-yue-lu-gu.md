# 月露谷 (Yue Lu Gu) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a 42-node, 25-character Stardew Valley-inspired lakeside town mod (zh-CN) under `configs/maps/yue-lu-gu/`.

**Architecture:** Standard agent-world mod — `manifest.json` + `map.json` + `characters/*.json`. No custom actions. Linear lakeshore layout: west bus-stop entry → main street commercial zone → lake/water zone → east shrine + firefly forest, with south-slope farm/ranch and scattered residences. Root node is conceptual (the valley itself), not the entry point.

**Tech Stack:** JSON config files validated by Zod schemas in `src/config/schemas.ts`. Validate with `npx tsx .claude/skills/agent-world-mod/scripts/validate.ts`.

**Design spec:** `docs/superpowers/specs/2026-05-05-yue-lu-gu-design.md`

**Reference files:**
- `src/domain/enums.ts` — NODE_TAGS, PROFESSIONS, OBJECTIVE_RELATION_KINDS, GENDERS, CHARACTER_ORIGINS
- `configs/maps/yu-no-tani/map.json` — example map structure
